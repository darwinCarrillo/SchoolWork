Please read the following to learn the usage of these scripts

Question 1:
>python tcomp1.py [MASTERFILE] [N(ngramSize)] [[CompareFile1] [CompareFile2] ...]
eg: >python tcomp1.py nm1.dat 2 nm2.dat

Question 2:
>python tcomp2.py [MASTERFILE] [[CompareFile1] [CompareFile2] ...]
eg: >python tcomp2.py tc1.dat tc2.dat tc3.dat

Question 3:
>python tcomp3.py [WORDFILE] [TEXTFILE] [INDEXFILE]
eg: >python index1.py word1.txt text1.txt index1.txt
